# For

```java
public class For {

 public static void main(String[] args) {
  for (int i = 0; i < 10; i++) {
   System.out.println("Hello moto " + i);
  }

 }

}
```

# while

```java
public static void main(String[] args) {
  int i = 0;
  while (i <10) {
   System.out.println("Hello moto " + i);
   i++;
  }

}

```

# do...while

```java
public static void main(String[] args) {
  int i = 0;
  do {
   System.out.println("hello " + i);
   i++;
  } while (i < 10);

 }
```
